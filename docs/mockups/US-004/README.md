# Mockup US-004 — Creare un nuovo portafoglio

Prototipo visivo per la user story US-004. Apri `index.html` in un browser.

## Schermate

| File | Descrizione |
|------|-------------|
| `index.html` | Vista principale: lista portafogli + form "Apri nuovo conto" |
| `errori.html` | Stati di validazione: nome vuoto e nome duplicato (toggle interattivo) |
| `successo.html` | Portafoglio appena creato visibile in lista, con banner di conferma |

## Navigazione

Le tre schermate sono collegate tra loro da link in fondo a ogni pagina.
`index.html` ha anche interazione JavaScript live: prova a registrare un conto vuoto
o con il nome «Conto Lungo Periodo» per vedere gli errori in azione.

## Stile visuale

Coerente con `docs/mockups/libro-mastro/` — registro contabile italiano degli anni Trenta:
carta manila, inchiostro ferrogallico, riga rossa di margine, font Playfair + Courier Prime.
